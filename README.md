# Head Start Modifier
Adds a Custom Run Modifier for getting a head start on card rewards and removals.

Start with 2 Strikes, 2 Defends, and your character's unique starters. Choose 9 out of 35 cards to add to your starting deck, like Sealed Deck. 5 of those 35 cards are Colorless.

## Installation
Unzip the contents of the .zip into the `mods` folder in the same location as `sts2.exe`. Requires [BaseLib](https://github.com/Alchyr/BaseLib-StS2/releases/latest).
